# 라이브러리 리스트

## express
### 서버 구축
### npm install express

## nodemon
### 소스 변경 감지 시 서버 자동 재기동
### npm install nodemon

## pm2
### 백그라운드에서 서버 기동(nodemon 과 연동하여 실행)
### npm install -g pm2

## better-module-alias
### import, require 등 에서 경로 지정 간소화
### npm install better-module-alias

## jsonwebtoken
### 사용자 인증 시 토큰 발급, 토큰 검증으로 인한 사용자 검증
### npm install jsonwebtoken
